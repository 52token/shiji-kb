# 史记特殊读音文档目录

本目录包含史记知识库项目中所有与特殊读音相关的文档。

## 📚 文档列表

### 1. 特殊读音词表维护说明.md
- **用途**: 特殊读音词典的维护规范和操作指南
- **内容**: 词条格式、添加流程、数据源标准
- **受众**: 维护人员

### 2. 特殊读音词表_正义版_说明.md
- **用途**: 基于史记正义的特殊读音词表说明
- **内容**: 从史记正义中提取的读音注释
- **受众**: 研究人员

### 3. 特殊读音注释提取结果.md
- **用途**: 自动提取特殊读音的实验结果
- **内容**: 提取方法、结果分析、质量评估
- **受众**: 技术人员

### 4. 特殊读音自动提取探索总结.md
- **用途**: 特殊读音自动提取项目的总结报告
- **内容**: 自动化提取探索、技术限制、改进建议
- **受众**: 项目组成员

### 5. 特殊读音发现方法.md
- **用途**: 系统化的特殊读音发现方法汇总
- **内容**: 五大发现方法、实施路线图、质量控制原则
- **受众**: 维护人员、研究人员

### 6. 更新日志_v2.4.md
- **用途**: v2.4版本更新说明
- **内容**: 新增14条词条明细、分类统计变化、质量控制标准
- **受众**: 项目组成员
