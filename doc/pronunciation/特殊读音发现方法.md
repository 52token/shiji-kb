#### 1.1 史记三家注

**史记正义**（唐·张守节）
- ✅ 已提取29条
- 特点：专门标注地名读音、反切注音
- 典型标注：`音焉与`、`楚庚反`
- **局限**：自动提取噪音率90%+，需人工筛选

**史记集解**（南朝·裴骃）

**史记索隐**（唐·司马贞）
- ⏳ 待深度挖掘

**操作建议**：
- 优先人工阅读"索隐"中的姓氏、人名部分
- 针对性提取匈奴、西域专有名词注音

#### 1.2 维基文库资源

**操作方法**:

### 四、半自动化提取法（改进版）

#### 4.1 种子词扩展法

**步骤**:
1. 从现有61条中提取关键字（如"句"、"华"、"乐"）
2. 全文检索史记，找出所有包含该字的专名
3. 逐个查史记三家注
4. 人工判断是否有特殊读音

**示例**:
```python
关键字: "句"
→ 检索结果: 句注、句践、句容、句章、句骊
→ 查注释: "句注音钩"、"句践音钩践"
→ 确认收录
```

#### 4.2 模式匹配 + 人工审核

**改进策略**（针对之前90%噪音问题）:
1. **缩小提取范围**：只提取人名、地名、官名后的注音
2. **提高模式精度**：
   - `[专名]，音[读音]`
   - `[专名]读如[读音]`
   - 韦昭/服虔/张守节注：`[专名]音[读音]`
3. **分级处理**：
   - 高置信度：直接收录
   - 中置信度：人工复核
   - 低置信度：丢弃

#### 4.3 专名列表交叉验证

## 质量控制原则

1. **来源可追溯**: 每条词条必须标注出处
2. **三家注优先**: 史记正义 > 史记集解 > 史记索隐
3. **权威书籍佐证**: 网络资源需有纸质书籍佐证
4. **专名优先**: 优先收录专有名词，谨慎收录常用多音字
5. **去重去噪**: 避免同义重复，排除通假字误判
