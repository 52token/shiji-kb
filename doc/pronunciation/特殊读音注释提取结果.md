# 《史记》特殊读音注释提取结果

## 概述

本文档从《史记集解三家注索隐正义》中提取特殊读音注释，重点关注地名、人名、姓氏等的古今读音差异。所有提取的字词均在《史记》原文中有实际出现。

**提取方法**：
- 反切注音（如"某音甲乙反"）
- 直接注音（如"某音某"）
- 通假注音（如"某读曰某"、"某读为某"）

**统计数据**：
- 总提取：1317条读音注释
- 去重后：699个不同字符
- 分类：地名558个、人名66个、姓氏32个、典籍/礼乐30个、其他13个

## 七、研究价值与应用

### 1. 古音研究价值
- **反切注音**: 保存了大量中古音韵资料
- **方言差异**: 反映南北方言读音差别
- **地名演变**: 记录古地名读音的历史变化

### 2. 文本标注应用
这些特殊读音可用于：
- 《史记》原文的注音标注
- 人名地名的规范化处理
- 实体识别的歧义消解
- 古籍阅读的辅助工具

### 3. 典型案例

#### 案例1：多音字"王"
- 平声（于方反）：君王、王者
- 去声（于放反）：霸王、称王

#### 案例2：地名"汜"
- 音祀：水名，在成皋
- 音凡：邑名，在襄城
- 音剑：水名，在定陶（高帝即位处）
- 音夷：桥名（楚人呼汜桥）

#### 案例3：姓氏"咎"
- 音高：咎如（赤狄别种）
- 不同于常见的"咎由自取"之"咎"

### 4. 注意事项
- 同一字在不同上下文可能有不同读音
- 部分读音为方言或专名特殊读音
- 通假字读音需根据原字义理解
