# 史记知识库 - 配置面板功能说明

## 功能概述

为所有 130 篇史记章节添加了右上角配置面板，允许用户自定义显示选项。

## 功能特性

### 1. 齿轮图标按钮
- **位置**：页面右上角固定位置
- **图标**：⚙️ (齿轮 emoji)
- **交互**：点击显示/隐藏配置面板

### 2. 配置面板
- **位置**：齿轮按钮下方
- **样式**：浮动白色卡片，带阴影和圆角
- **关闭方式**：
  - 点击面板外部
  - 按 ESC 键
  - 再次点击齿轮图标

### 3. 语法高亮开关
- **默认状态**：开启
- **开启效果**：显示所有实体的彩色标注、下划线、背景色等
- **关闭效果**：纯文本显示，移除所有语法高亮效果
- **状态持久化**：使用 localStorage 保存用户选择

## 技术实现

### 修改的文件

1. **CSS 样式** - `docs/css/shiji-styles.css`
   - 配置按钮和面板样式
   - 语法高亮关闭状态的样式重置
   - 覆盖所有实体类型（人名、地名、官职、时间等）

2. **JavaScript 脚本** - `docs/js/settings-panel.js`
   - 面板切换逻辑
   - 复选框事件监听
   - localStorage 状态管理
   - 键盘快捷键支持

3. **HTML 章节文件** - `docs/chapters/*.html`
   - 所有 131 个文件（130 章 + index.html）
   - 在 `<body>` 后添加配置面板 HTML
   - 在 `</body>` 前引入 JavaScript 脚本

### 批量处理脚本

- **添加脚本**：`scripts/add_settings_panel.py`
  - 批量为所有章节添加配置面板
  - 自动检测是否已添加，避免重复

- **验证脚本**：`scripts/verify_settings_panel.py`
  - 验证所有文件是否正确添加配置面板
  - 检查关键元素和脚本引用

## 使用方法

1. 打开任意史记章节页面
2. 点击右上角的齿轮图标 ⚙️
3. 在弹出的配置面板中：
   - 勾选"语法高亮"：显示彩色标注
   - 取消勾选"语法高亮"：纯文本显示
4. 配置会自动保存，下次打开页面时保持

## 扩展性

配置面板的结构设计为易于扩展，未来可以添加更多选项，例如：
- 字体大小调节
- 行间距调节
- 实体类型过滤（只显示特定类型的高亮）
- 主题切换（日间/夜间模式）
- Purple Number 显示开关
