# Ontology v2（实验版本）

> **版本说明**：这是实验版本，用于知识覆盖度转化相关实验。稳定版本请参考 [ontology-v1](../ontology-v1/)。

## 实验目标

本版本主要用于以下实验：

1. **知识覆盖度转化**：探索知识单元如何更有效地覆盖原文内容
2. **新的知识组织方式**：测试不同的 SKU 结构和分类方法
3. **覆盖度分析方法**：开发和验证新的覆盖度评估工具
