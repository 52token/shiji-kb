## 矛盾类型分类

基于SKILL_07的矛盾分类体系:

| 类型 | 定义 | 已发现案例 |
|------|------|----------|
| **DATE** | 同一事件记载不同时间 | 陈涉起义时间、太子丹之死、下宫之难 |
| **DETAIL** | 数量/地点/参与者不一致 | 东城斩首数、攻城阳战役地点 |
| **ATTRIBUTION** | 责任/功劳归属不同 | 函谷关守关者、鸿门宴核心推动者 |
| **SEQUENCE** | 事件顺序或因果链不一致 | 下宫之难日期与因果链 |

## 反常类型分类

基于SKILL_07c的反常分类体系:

| 类型 | 定义 | 已发现案例 |
|------|------|----------|
| **数字反常** | 违反人口/兵力/年龄/财富常识 | 长平之战40万降卒、东城斩首8万 |
| **制度反常** | 违反爵位/继承/礼制规范 | 吕后称制、秦始皇无皇后 |
| **叙事反常** | 应有记载缺失(沉默异常) | 秦始皇无皇后记载 |
| **年龄反常** | 超出生理/从政/征战年龄限制 | 廉颇八十余岁仍欲为将 |
| **行为反常** | 违反生理/社会常规 | 项羽力能扛鼎 |

### 模式发现

详见 [anomaly_detection_2026-03-26.md#五-模式发现](anomaly_detection_2026-03-26.md#五-模式发现)

1. **数字反常模式**: 数量级差异常见,视角影响显著,夸大系数约2-5倍
2. **制度反常模式**: 多发生在制度转型期/权力真空期,通常不可持久
3. **时间矛盾模式**: 差异<3年可能是纪年方式,>10年可能是史源差异

### 矛盾检测流程

详见 [contradiction_detection_methodology.md](contradiction_detection_methodology.md)

```
事件索引A + 事件索引B
    ↓ 事件对齐(公元纪年 + 实体共现 + 关键词匹配)
重叠事件对列表
    ↓ 逐维度对比(时间/地点/人物/数量/归因/顺序)
差异列表
    ↓ 分类(真矛盾 vs 视角差异 vs 详略不同)
矛盾报告
```

## 矛盾与反常的关联

矛盾往往揭示反常,反常往往导致矛盾。详见 [contradiction_to_anomaly_mapping.md](contradiction_to_anomaly_mapping.md)

**典型关联**:
- **数量矛盾** → 数字反常 (如东城斩首: 百人 vs 八万)
- **时间矛盾** → 时间异常 (如下宫之难: 相差14年)
- **归因矛盾** → 叙事反常 (如函谷关守关者归属不明)
- **沉默矛盾** → 叙事反常 (如秦始皇无皇后记载)
