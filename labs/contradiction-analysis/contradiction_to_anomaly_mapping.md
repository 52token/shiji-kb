# 矛盾与反常的关联映射

> 矛盾往往揭示反常，反常往往导致矛盾。本文档建立两者之间的系统性关联。

---

## 一、核心理念

**矛盾检测 (SKILL_07)** 和 **反常推理 (SKILL_07c)** 是一体两面：

```
矛盾检测:  两处记载互相打架  →  "A说X，B说Y，两者矛盾"
反常推理:  单条记载不合常理  →  "A说X，但X违反常识"

两者的交集:
  如果A说X，B说Y，且X或Y本身就不合常理
  →  这既是矛盾，也是反常
  →  高价值研究线索
```

---

## 二、矛盾类型 → 反常类型映射

### 2.1 DATE矛盾 → 时间异常

**映射规则**：
```
IF 两处记载的时间差异 > 阈值:
    矛盾类型 = DATE
    反常类型 = 时间异常 (SKILL_07c: 数字反常/时间异常)
```

**已发现案例**：

| 矛盾案例 | 时间差 | 反常类型 | 研究价值 |
|---------|--------|---------|---------|
| **下宫之难** (039 vs 043) | 14年 | 时间异常（差异过大） | 高 |
| **太子丹之死** (086 vs 006) | 4年 | 时间异常（可能纪年方式不同） | 中 |
| **陈涉起义** (007 vs 008) | 七月 vs 秋 | 细微差异（非异常） | 低 |

**分析思路**：
- 时间差>3年 → 可能存在严重史源问题
- 时间差1-3年 → 可能是纪年方式差异（如秦历vs汉历）
- 时间差<1年（如月份差异） → 可能是精确度不同

---

### 2.2 DETAIL矛盾（数量） → 数字反常

**映射规则**：
```
IF 两处记载的数字差异 > 10倍:
    矛盾类型 = DETAIL (数量矛盾)
    反常类型 = 数字反常 (SKILL_07c: 兵力异常/财富异常)
```

**已发现案例**：

| 矛盾案例 | 数量对比 | 差异倍数 | 反常类型 | 可能解释 |
|---------|---------|---------|---------|---------|
| **东城斩首** (007 vs 008) | 数十百人 vs 八万 | 80:1 | 兵力异常 | 统计口径不同（单次战斗 vs 战役全程） |
| **垓下参战力量** (007 vs 008) | 多方汇聚 vs 三十万 | - | 描述方式不同 | 007强调多方，008强调韩信主力 |

**SKILL_07c反常推理的补充**：
- 即使没有矛盾，"八万斩首"本身也是数字反常
- 反常推理会问："单次战役斩首八万是否可能？"
- 矛盾检测会问："为什么007和008记载的数字差异如此大？"

**联合分析框架**：
```
SKILL_07矛盾检测 → 发现"数十百人 vs 八万"的矛盾
    ↓
SKILL_07c反常推理 → 识别"八万斩首"为数字反常
    ↓
SKILL_07e真实性推理 → 科学鉴定+动机分析 → 可信度C级
    ↓
结论: 核心事实可信，数字可能夸大，统计口径不同
```

---

### 2.3 ATTRIBUTION矛盾 → 叙事反常

**映射规则**：
```
IF 两处记载的行为主体不同 OR 归因不同:
    矛盾类型 = ATTRIBUTION
    反常类型 = 叙事反常 (SKILL_07c: 详略失衡/评价矛盾)
```

**已发现案例**：

| 矛盾案例 | 归因对比 | 反常类型 | 分析 |
|---------|---------|---------|------|
| **函谷关守关者** (007 vs 008) | 关守卒 vs 沛公 | 详略失衡 | 007淡化刘邦主动性，008强调刘邦主动封关 |
| **鸿门宴核心推动者** (007 vs 008) | 范增/项庄 vs 张良/樊哙 | 视角差异 | 007项羽阵营视角，008刘邦阵营视角 |

**这是矛盾还是反常？**
- **矛盾检测视角**：两处记载的归因主体不同
- **反常推理视角**：详略失衡/叙事重心差异
- **真实性推理视角**：可能都是真的，只是视角不同

**结论**：归因矛盾往往不是"事实矛盾"，而是"叙事视角差异"。

---

### 2.4 SEQUENCE矛盾 → 因果链断裂

**映射规则**：
```
IF 两处记载的事件顺序不同 OR 因果链不同:
    矛盾类型 = SEQUENCE
    反常类型 = 动机缺失 / 后续缺失 (SKILL_07c: 行为反常)
```

**已发现案例**：

| 矛盾案例 | 顺序对比 | 反常类型 | 分析 |
|---------|---------|---------|------|
| **下宫之难** (039 vs 043) | 前583年直接诛杀 vs 前597年诛杀→15年→复位 | 因果链完整性不同 | 043的因果链更完整（灭族→救孤→隐匿15年→复位） |

**SKILL_07c反常推理的补充**：
- 如果只看039晋世家："前583年直接诛杀赵同赵括，然后复立赵武"
  - 反常点：为什么诛杀后又立即复立？缺乏动机解释
- 如果只看043赵世家："前597年诛杀，隐匿15年后复位"
  - 更合理：有救孤、隐匿、韩厥进言的完整因果链

**联合分析**：
```
矛盾检测 → 发现时间和因果链矛盾
    ↓
反常推理 → 039版本缺乏动机解释（反常）
    ↓
真实性推理 → 043版本因果链更完整 → 可信度更高
```

## 四、矛盾→反常→真实性的分析流程

### 4.1 高价值案例：东城斩首

**第一步：矛盾检测 (SKILL_07)**
```
发现: 007记载"数十百人"，008记载"八万"
类型: DETAIL矛盾（数量）
差异: 80:1
```

**第二步：反常推理 (SKILL_07c)**
```
问题1: "八万斩首"本身是否合理？
    → 数字反常：单次战役斩首八万，超出常规
    → 研究价值：高

问题2: 有无通常的历史学解释？
    → 解释1：统计口径不同（单次战斗 vs 战役全程）
    → 解释2：007聚焦项羽个人，008聚焦汉军战果
    → 解释3：数字夸大（古代战报惯例）
```

**第三步：真实性推理 (SKILL_07e)**
```
方法1: 科学鉴定
    → 项羽28骑突围，汉军数千骑追击
    → 单次斩杀八万不符合兵力规模

方法2: 动机分析
    → 汉军有夸大战果动机
    → 灌婴家族有传承动机

方法3: 逻辑自洽性检验
    → 007和008统计口径不同，都可能是真的

可信度评级: C级（核心事实可信，数字可能夸大）
```

**结论**：
- 矛盾：两处记载数字不一致
- 反常：八万斩首本身是数字反常
- 真实性：核心事实（项羽败亡）可信，具体数字存疑

---

### 4.2 高价值案例：下宫之难

**第一步：矛盾检测**
```
发现: 039记前583年，043记前597年（差14年）
类型: DATE矛盾 + SEQUENCE矛盾
```

**第二步：反常推理**
```
问题1: 039版本的反常点
    → 039: 前583年诛杀赵同赵括，然后立即复立赵武
    → 反常: 为什么诛杀后又立即复立？缺乏动机解释

问题2: 043版本的因果链
    → 043: 前597年屠岸贾诛杀→程婴救孤→隐匿15年→前583年复位
    → 合理: 因果链完整，动机清晰
```

**第三步：真实性推理**
```
方法1: 沉默证据法
    → 039未提屠岸贾、程婴救孤
    → 可能原因：039视角为晋国官方，赵氏内部事务简化

方法2: 逻辑自洽性检验
    → 043的因果链更完整
    → 039的简化版本缺乏逻辑连贯性

可信度评级:
    → 043版本: B级（因果链完整，较可信）
    → 039版本: C级（可能简化了复杂过程）
```

**结论**：
- 矛盾：时间和因果链都不一致
- 反常：039版本缺乏动机解释
- 真实性：043版本更可信

## 五、反常但无矛盾的案例

有些反常记载并不构成矛盾，因为只在一个章节中出现。

| 反常案例 | 章节 | 反常类型 | 是否有矛盾？ | 研究价值 |
|---------|------|---------|------------|---------|
| **秦始皇无皇后** | 006 | 叙事反常：沉默异常 | 无（只在006） | 高 |
| **白起坑杀40万** | 073 | 数字反常：兵力异常 | 无（只在073） | 高 |
| **吕后称制** | 009 | 制度反常：女性执政 | 无（只在009） | 中 |
| **项羽力能扛鼎** | 007 | 数字反常：夸张描写 | 无（只在007） | 低 |

**分析思路**：
- 这些反常没有跨章节矛盾，但本身值得研究
- SKILL_07c可以单独识别这些反常
- SKILL_07e可以对它们进行真实性推理（无需矛盾触发）

## 六、工作流程总结

### 6.1 从矛盾到反常（矛盾触发）

```
矛盾检测 (SKILL_07)
    ↓ 发现跨章节矛盾
    ↓
检查: 矛盾中是否包含反常？
    ↓ IF 数量差异>10倍 OR 时间差异>3年 OR 归因缺失
    ↓
反常推理 (SKILL_07c)
    ↓ 标注反常类型
    ↓ 评估研究价值
    ↓
真实性推理 (SKILL_07e)
    ↓ 深度分析
    ↓
可信度评级
```

### 6.2 从反常到矛盾（反常触发）

```
反常推理 (SKILL_07c)
    ↓ 扫描全书，发现单条反常记载
    ↓
检查: 这个事件在其他章节有无记载？
    ↓ IF 有
    ↓
矛盾检测 (SKILL_07)
    ↓ 对比记载是否一致
    ↓ IF 不一致
    ↓
发现矛盾 + 反常的高价值案例
```

### 6.3 两者并行（批量扫描）

```
全书扫描
    ├─ SKILL_07 矛盾检测 → 输出矛盾清单
    └─ SKILL_07c 反常推理 → 输出反常清单
        ↓
交叉匹配（矛盾 ∩ 反常）
    ↓
高价值研究线索
    ↓
SKILL_07e 真实性推理
    ↓
《史记可信度与谣言图谱》
```
